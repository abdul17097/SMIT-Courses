Types of Attribute Selectors [attr] — Attribute Presence Selector Selects elements that have the specified attribute, regardless of its value. Example: css Copy code [title] { color: blue; } html Copy code
This text will be blue.
This text won't be blue.
[attr=value] — Exact Match Selector Selects elements with an attribute that exactly matches the given value. Example: css Copy code [type="button"] { background-color: yellow; } html Copy code [attr~=value] — Word Match Selector Selects elements with an attribute containing a word that matches the value, separated by spaces. Example: css Copy code [class~="featured"] { border: 2px solid green; } html Copy code

[attr|=value] — Prefix Match Selector Matches elements with an attribute value that is exactly the given value or starts with the value followed by a -. Example: css Copy code [lang|="en"] { font-style: italic; } html Copy code

This will not be italic.

[attr^=value] — Starts With Selector Selects elements with an attribute value that begins with the specified value. Example: css Copy code [href^="https"] { color: green; } html Copy code Secure Link Non-Secure Link [attr$=value] — Ends With Selector Selects elements with an attribute value that ends with the specified value.
